Client Dashboard — Intake & New Client Workflow

How to record a prospective client, start a new client service, collect payment details, and trigger contract + invoicing.

Training lessons

Lesson 1 — Create / Locate Client Intake

Key idea: Enter the prospective client's contact and demographic data in the Intake form before you can create a new client dashboard entry.

Step-by-step:

  1. Open the Intake page from the dashboard menu.
  2. Fill out required fields: full name, DOB, place of birth, contact info, and any immediate notes.
  3. Save the intake — the record will appear in the intake list (use this list when creating the new client dashboard entry).

Screenshot: intake list and sample intake form (example).

Lesson 2 — Start New Client Dashboard Entry

Key idea: When the prospect retains the firm, open the New Client Dashboard form and select the intake you created.

Step-by-step:

  1. Navigate to New Client Dashboard Form from the dashboard menu.
  2. From the client selector, choose the intake record you previously saved.
  3. Confirm name, contact, and any intake notes are populated into the form.

Screenshot: new client dashboard form showing client selector and case-type dropdown.

Lesson 3 — Select Case / Petition Type

Key idea: Choose the application type(s) (e.g., I-130, I-589, waivers) from the list. This drives subsequent fields and fees.

Step-by-step:

  1. Open the Petition Type dropdown and select the desired application(s).
  2. If the selected type requires additional details (e.g., A-numbers), click "Add record" or complete the prompted fields.
  3. Confirm the case status and any special flags (e.g., asylum = yes/no).

When multi-party applications are selected, additional A-number fields will appear for dependents.

Lesson 4 — Add Dependents / A-numbers

Key idea: If the application includes spouses/dependents, populate their A-numbers in the extra fields.

Step-by-step:

  1. Click the A-Number field to add the client's A-number (if available).
  2. If the application includes additional people, populate each additional A-number field that appears.
  3. Confirm each dependent’s name and A-number are correct before saving.

Example layout for adding A-numbers and dependents.

Lesson 5 — Payment Method & Plans

Key idea: Choose how the client will pay today (cash/card) or which payment plan they choose; offer automatic payments enrollment for the discount.

Step-by-step:

  1. Locate the "Initial Payment Method" field and select the method (Cash / Card / Bank / Other).
  2. If selecting a payment plan, choose the plan amount from the Fee Agreement dropdown.
  3. Record the staff member who read the fee agreement in the client's primary language.
  4. Ask client if they want automatic payments (discount applies); select the automatic payments option if they do.

Payment method and fee agreement fields example.

Lesson 6 — Submit, Contract & Apply Payment in Docketwise

Key idea: After submitting the New Client Dashboard, the system generates a contract and a Docketwise account/invoice; then apply payment in Docketwise.

Step-by-step:

  1. Click Submit. The client receives an emailed contract to sign electronically.
  2. Open Docketwise → Client → Invoice for the newly created matter.
  3. Choose the payment type in Docketwise (credit card, cash, etc.), select correct office account/date/amount, and save payment details.
  4. If paying by card, enter billing info and finalize the payment to apply funds to the invoice.

Quick Reference

Before you start: Intake must be created and saved. (See Lesson 1 screenshot.)

Important fields:
- Client selector (use saved intake)
- Petition Type / Add record for immigration court cases
- A-numbers for dependents
- Fee Agreement / Automatic payments

After submit: Check client email for contract; open Docketwise invoice to apply payment.
